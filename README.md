# world-population-analysis-

## Description
Personal data analysis project focused on analyzing world population data using Python and SQL 

## Objectives
- Analyze population growth trends across different countries
- Compare population distribution by continent
- Identify countries with the highest and lowest population growth

## Planned Steps
1. Collect public population datasets from reliable sources
2. Clean and prepare the data using Python (Pandas)
3. Perform basic statistical analysis and aggregations
4. Use SQL queries to explore population trends
5. Create simple charts and tables using Google Sheets or Tableau

## Tools & Technologies
- Python (Pandas/NumPy)
- SQL
- Google Sheets
- Tableau (basic)
- Statistics
